# Homebrew PorkyHUD

Homebrew tap for [PorkyHUD](https://github.com/Mengjiehasaheart/PorkyHUD), a terminal-first Apple Silicon system HUD for macOS.

## Install

```bash
brew tap Dr-Matcha-Studio/porkyhud
brew install porkyhud
porkyhud
```

One-command install:

```bash
brew install Dr-Matcha-Studio/porkyhud/porkyhud
```

Development build:

```bash
brew install --HEAD Dr-Matcha-Studio/porkyhud/porkyhud
```

PorkyHUD runs with normal user permissions for most metrics. Advanced CPU/GPU power and residency counters use macOS-protected tooling and may ask for administrator access inside the app.
